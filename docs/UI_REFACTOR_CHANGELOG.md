## 🔄 流 Процес роботи

### Сценарій 1: Стандартний (файл в папці справи)

```
1. Користувач вибирає папку справи
2. Система шукає search_phrases.txt в цій папці
3. Якщо знайдено → активує кнопку
4. Користувач нажимає "Запустити"
5. Система нарізає за цим файлом
```

### Сценарій 2: Окремий файл (з іншого місця)

```
1. Користувач вибирає папку справи
2. Користувач включає чекбокс "Вибрати окремий файл"
3. Користувач вибирає файл через file picker
4. Система перевіряє файл
5. Якщо все ОК → активує кнопку
6. Користувач нажимає "Запустити"
7. Система нарізає за вибраним файлом
```
