## 📄 Подготовка файла фраз

Создайте файл `search_phrases.txt` в папке дела:

```
1 батько бачив це особисто
08:45---08:55

2 свідок підтверджує алібі
15:30---15:45

3 прокурор припускає без доказів
22:15---22:25
```

**Правила:**
- Одна фраза на рядке + время внизу
- Формат времени: `MM:SS---MM:SS`
- Можно в одном рядке: `фраза (10:30---10:45)`

## 🎯 Результаты

После нажатия кнопки система создаст:

```
case_folder/
└── _CourtDefense/
    └── 02_нарізки_за_мітками/
        ├── батько_бачив__recording_min_8-45/
        │   ├── нарізка.mp3 (10 секунд)
        │   ├── ШІ_АНАЛІТИКА_ФРАГМЕНТА.txt
        │   └── фрагмент_контексту.txt
        ├── свідок_підтверджує__recording_min_15-30/
        │   └── ...
        └── прокурор_припускає__recording_min_22-15/
            └── ...
```

**Что в каждой папке:**
- 🎵 **нарізка.mp3** — вырезанный фрагмент
- 💬 **фрагмент_контексту.txt** — текст ±15 сек от момента
- 🤖 **ШІ_АНАЛІТИКА_ФРАГМЕНТА.txt** — анализ Claude

## ⚡ Примеры

### Выделить показания свидетелей

```

02:15---02:30
```

→ Автоматически вырезаны 3 фрагмента с анализом

```

25:00---25:20
```

→ Выделены спекулятивные высказывания

## 🛠️ Решение проблем

| Проблема | Решение |
|----------|---------|
| "Файл не найден" | Создайте `search_phrases.txt` в папке дела |
| "FFmpeg не найден" | Скачайте с https://ffmpeg.org/download.html |
| "Папка не доступна" | Проверьте права доступа на чтение |
| "Нет результатов" | Проверьте наличие .mp3/.wav файлов |
