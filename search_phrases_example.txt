# ПОРАДИ ПО ЧАСОВИМ МІТКАХ:
# ═════════════════════════════════════════════════════════════════════════════
#
# 1. Як знайти правильний час?
#    • Слухайте запис з програмою типу VLC (MediaPlayer)
#    • Там буде показано точний час
#    • Позначайте час ПОЧАТКУ фрази
#
# 2. Щоб дізнатися час фрази:
#    • Якщо у вас є JSON файл з Whisper (транскрипція)
#    • В ньому буде: "start": 8.45, "end": 8.95
#    • Перевести в MM:SS: 8.45 = 00:08:45
#
# 3. Як обрати інтервал?
#    • start = час ПОЧАТКУ фрази
#    • end = час КІНЦЯ фрази + 1-2 сек для контексту
#    • Приклад: "свідок каже" = 10:30 - "каже" закінчується 10:45
#    • Пишіть: 10:30---10:45
#
# 4. Кількість фраз:
#    • Можете додати стільки, скільки потрібно
#    • Немає ліміту!
#
